# FastTrack

세종 도심권 위치 관제 웹 프로토타입입니다.

## 주요 기능

- OpenStreetMap(Leaflet) 기반 지도 표시
- 시간 순 위치 데이터 동선 시각화
- 10분 이상 체류 구간 자동 감지
- 체류 시간 구간별 원 크기/색상 차등 표시
- 최신 위치(초록 점멸) 표시
- 수동 문자 입력(통신사 포맷/CSV) 기반 동선 재구성
- 이동 상태 아이콘 표시(차량/대중교통 추정)

## 실행 방법

1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.
2. 우측 입력 패널에서 샘플 데이터 또는 사용자 데이터를 넣고 `동선 반영`을 누릅니다.

## 파일 구성

- `index.html`: UI 레이아웃
- `styles.css`: 스타일 정의
- `app.js`: 지도 렌더링, 체류 분석, 입력 파싱 로직

## 주의

- 본 프로젝트는 프로토타입이며 실제 업무 적용 전 법적/보안 검토가 필요합니다.
- 외부 지오코딩/라우팅 API 응답 상태에 따라 일부 표시가 달라질 수 있습니다.
